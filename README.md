# Microscope
## Eindopdracht Meteor SRP

Deze app maakt het mogelijk om specifieke websites te delen, en te bespreken met andere gebruikers. Via de categoriën bovenaan kun je (na toevoegen/'liken') de verschillende posts filteren. Dit alles is mogelijk zodra je een account hebt, en in bent gelogd.

### Functionaliteiten
Door het toevoegen van het account, wordt je direct ingelogd en kun je verder met het gebruik van de applicatie. Als niet gebruiker kun je enkel bekijken.

Websites toevoegen is mogelijk via de Submit functie. Daar wordt gecontrolleerd of er überhaupt wat is ingevuld op het moment van verzenden, en of de website al eerder is toegevoegd.

Like een website als je deze leuk vindt. Zo krijgt deze een score.
